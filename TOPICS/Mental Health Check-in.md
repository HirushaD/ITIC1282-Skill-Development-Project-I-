# Mental Health Check-in

## Project Type
Web Application (HTML + CSS + JavaScript + Backend)

## What It Is
A daily mood tracker where students log their emotional state anonymously. Aggregated data shows campus-wide mood trends without identifying individuals. Includes simple self-help resources.

## Core Features
- **Daily Check-in** — select mood from 5 levels (Very Bad / Bad / Neutral / Good / Very Good)
- **Optional Note** — brief text note about how you're feeling (anonymous)
- **Trend Dashboard** — see your own mood history as a line chart
- **Campus Aggregated Insights** — anonymous campus-wide mood trends (e.g. "60% of students feel stressed before exams")
- **Resource Library** — static self-help content: breathing exercises, study tips, counselor contacts
- **Anonymous Journal** — write private journal entries (not shared)
- **Weekly Email Summary** — opt-in digest of personal mood trends
- **No Login Required** — use a generated anonymous ID to track personal history

## Similar Existing Platforms
- **eMoods** (emoodtracker.com) — mood tracking with bipolar disorder focus
- **MindDoc** — guided mental health check-ins
- **Mindfit / Moodfit** — mood and habit tracking with meditation
- **IIWIARS** — anonymous mental health venting platform

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js + Express or Python Flask
- Database: SQLite
- Charts: Chart.js for mood history visualization

## Key Modules
1. **Anonymous ID Generator** — creates a random ID to track individual history without login
2. **Check-in Module** — mood selection + optional note
3. **Personal Trend Module** — individual mood chart
4. **Aggregated Insights Module** — campus-wide anonymous statistics
5. **Resource Library** — static self-help pages
6. **Journal Module** — private anonymous entries

## Privacy Considerations
- No identifying information collected
- Aggregated data never shows individual entries
- Clear data retention policy (e.g. delete entries older than 1 year)

## Project Complexity
- Low-Medium — straightforward logging + chart visualization
- Achievable for 3-4 members in one semester

## Unique Angle (Sri Lankan Context)
- Local counselor contact numbers for Sri Lankan universities
- Sinhala/Tamil mood labels
- Campus-specific stress triggers (e.g. exam season, ragging)
- Integration with university counseling services
