# Open Mic Night Sign-up

## Project Type
Web Application (HTML + CSS + JavaScript + Backend)

## What It Is
A web-based sign-up system for managing open mic or open mic night events. Handles performer registrations, waitlists, and event schedules automatically.

## Core Features
- **Event Creation** — create event with date, time, venue, description, theme, max performers
- **Performer Registration** — sign up with name, act type (music/speech/comedy/poetry), slot preference
- **Auto-Scheduling** — assign performers to time slots automatically
- **Waitlist Management** — if slots fill, others join waitlist; auto-promote when slot opens
- **Event Schedule Display** — public page showing full lineup and times
- **Check-in System** — host marks performers as arrived/performed
- **Performer Notes** — equipment needs, special requests per act
- **Email/SMS Reminder** — send reminders to registered performers before event

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js + Express or Python Flask
- Database: SQLite

## Key Modules
1. **Event Module** — create, edit, delete events
2. **Performer Sign-up Module** — register with name, act type, preferences
3. **Auto-Scheduler** — assign time slots by signup order or preference
4. **Waitlist Engine** — manage overflow and auto-promotion
5. **Public Schedule Page** — real-time lineup display
6. **Check-in Module** — host marks performer attendance
7. **Notification Module** — email or SMS reminders

## Project Complexity
- Low-Medium — form handling + scheduling logic + notifications
- Very achievable for 2-3 members in one semester

## Unique Angle (Sri Lankan Context)
- Support for local university cultural events (e.g. Drama Society, Music Society)
- Sinhala/Tamil act names and descriptions
- WhatsApp group link generation for performers
- Integration with local venue maps
